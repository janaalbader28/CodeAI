## How to Use

1. Open the website: [https://codeai-v1.netlify.app](https://codeai-v1.netlify.app)  
2. Click **Get Extension** to download the Chrome extension zip file  
3. Extract the downloaded folder  
4. Open Chrome and go to `chrome://extensions/`  
5. Enable **Developer Mode**  
6. Click **Load unpacked** and select the extracted folder  
7. Use the website and extension together for AI-powered code help
